# BAC PPT Presentation Risk Rules

This file converts the Phase 5 numeric and basis validation into slide-language rules.

Use these rules while drafting slides so the deck remains credible, academically defensible, and internally consistent.

## 1. Overall Framing Rule

The BAC PPT should present the design as:
- a feasibility-grade academic plant design
- based on a selected BAC process route and commercial product basis
- supported by internally consistent screening calculations
- not a bankable EPC-level or fully de-risked industrial package

Preferred phrasing:
- "feasibility-grade design basis"
- "screening-level economic estimate"
- "model-based financial indicators"
- "preliminary equipment sizing basis"

Avoid:
- "final design"
- "bankable economics"
- "proven commercial guarantees"
- "fully validated industrial package"

## 2. Reactor Wording Rule

Use only this reactor wording in slides:

`Glass-lined agitated vertical reactor with external recirculation heat-removal loop`

Why:
- the reactor chapter explicitly states that the glass-lined external-loop basis supersedes older jacketed-reactor wording
- raw artifact wording still contains `Jacketed CSTR Train`, which should not appear in the PPT

Allowed supporting phrasing:
- "GLR external-loop basis"
- "external pump-around heat removal"
- "preliminary mechanical closure tied to the reactor appendix"

Avoid:
- "jacketed CSTR train"
- mixing both descriptions across different slides

## 3. Route-Selection Wording Rule

When discussing the selected route, use:
- "selected on a screening-feasible basis"
- "preferred route under the current technical and economic screening"
- "selected route for feasibility-grade design development"

Avoid:
- "definitively proven best route"
- "fully optimized industrial route"

Recommended slide line:

`The solvent-free quaternization route is selected as the current feasibility-grade design basis because it gave the strongest combined process and economic screening score under the present assumptions.`

## 4. Economics Wording Rule

### 4.1 High-Risk Metrics

The following metrics are allowed in the deck, but only with softening language:
- payback
- IRR
- NPV
- gross margin
- break-even capacity

### 4.2 Preferred Labeling

Use these labels:

| Metric | Preferred Label |
| --- | --- |
| Payback | Model-derived payback period |
| IRR | Screening-level IRR |
| NPV | Model-based NPV |
| Gross margin | Gross margin on current selling-price basis |
| Break-even capacity | Break-even capacity under current assumptions |

### 4.3 Required Economics Caveat

At least one economics slide should contain a note like:

`These financial indicators are model-based feasibility outputs and remain sensitive to route assumptions, purification performance, and selling-price basis.`

## 5. Working-Capital Wording Rule

Working capital must be presented as:
- a cash-cycle model result
- linked to inventory, receivable, payable, and pre-commissioning assumptions

Avoid saying:
- "working capital is simply 7 / 15 / 30 days of operating cost"

Preferred wording:

`Working capital was estimated from the modeled inventory, receivable, payable, and pre-commissioning cash-cycle basis rather than from a single blanket percentage.`

## 6. CAPEX Wording Rule

Project cost and equipment cost should be described as:
- preliminary project cost estimate
- screening-level CAPEX estimate
- equipment-family cost estimate

Because:
- reactor pricing still carries a legacy pressure-vessel note pending full GLR repricing

Recommended note for the project-cost slide:

`The reported CAPEX values are screening-level estimates intended for feasibility assessment and should be refined further during detailed design.`

## 7. Product-Form Wording Rule

Because carrier wording is not fully reconciled, use:
- `50 wt% BAC solution`
- `commercial sold-solution basis`

Avoid:
- detailed carrier composition wording unless it is fully reconciled
- switching between "aqueous", "water + ethanol", and "water-only" without explanation

## 8. Site-Selection Wording Rule

Use:
- `Dahej, Gujarat is selected as the preferred site basis`
- `selected for the current India feasibility case`

Avoid:
- implying an exhaustive artifact-based numerical comparison if only one candidate survives in the current structured output

Recommended wording:

`Dahej is used as the preferred site basis because it offers the strongest combined logistics, utility, regulatory, and chemical-cluster fit for the current BAC case.`

## 9. Slide-Specific Caution Rules

| Slide | Rule |
| --- | --- |
| Reactor slides | use only the glass-lined external-loop reactor description |
| Process-selection slide | keep `screening-feasible` language visible |
| CAPEX slide | call costs preliminary or screening-level |
| OPEX slide | note that raw materials dominate the cost structure |
| Financial slide | include one caveat sentence about sensitivity to assumptions |
| Conclusion slide | say "technically coherent and economically promising under current assumptions" rather than "fully validated and viable" |

## 10. Recommended Slide-Safe Sentences

These are ready-to-use lines for the deck.

### Reactor Slide

`R-101 is presented on the adopted glass-lined, externally cooled reactor basis selected for safe preliminary handling of the exothermic quaternization service.`

### Process Selection Slide

`The solvent-free route is retained as the preferred feasibility-grade route because it scored highest under the current technical and economic screening basis.`

### Project Cost Slide

`Project cost has been estimated on a screening basis to support feasibility assessment; further refinement is required at detailed-design stage.`

### Working Capital Slide

`Working capital is derived from the modeled operating cash cycle, including inventory, receivables, payables, and pre-commissioning provisions.`

### Financial Slide

`The reported payback, IRR, and NPV should be read as model-based feasibility indicators tied to the present process and market assumptions.`

### Conclusion Slide

`Overall, the BAC plant appears technically coherent and economically promising on the current feasibility basis, subject to further route, purification, and detailed-cost refinement.`

## 11. Numbers To Keep But Not Over-Emphasize

These values may appear in tables, but should not all be repeated verbally as headline claims.

| Value | Handling Rule |
| --- | --- |
| Payback = 0.48 y | show in table, but qualify verbally |
| IRR = 60.00% | show in table, but avoid celebratory tone |
| NPV = 4,366.57 Cr INR | show in table, but mention assumption sensitivity |
| Gross margin = 65.87% | show as part of model outcome, not as a guaranteed margin |
| Break-even = 34.10% | acceptable summary metric |

## 12. Final Rule

If a slide statement sounds stronger than the source basis, soften it.

For this deck:
- choose precision over hype
- choose consistency over novelty
- choose defensible engineering language over optimistic sales language
