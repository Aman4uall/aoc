# Product Profile Note

para-Nitroanisole is treated in this example as an organic specialty intermediate used in dyes, pigments, and related synthesis chains.

Key properties recorded for the seeded workflow:
- Molecular weight: 153.14 g/mol
- Melting point: about 54 C
- Boiling point: about 260 C
- Density: about 1.17 g/cm3

The plant-design framing assumes a high-purity finished product and a moderate annual production scale suited to specialty-chemical manufacture rather than commodity production.
